# 07 — Troubleshooting

## Агент сразу делает картинки и не задаёт вопросы

Проблема: нет brand intake.

Исправление:

```text
Остановись. Сначала собери brand-profile. Не создавай финальную карусель до brand intake.
```

## Карусель выглядит красиво, но бессмысленно

Проблема: визуал сделан раньше маркетингового brief.

Исправление:

```text
Вернись к SMM brief. Для каждого слайда напиши meaning job и why swipe/save.
```

## Слишком много текста на слайдах

Исправление:

```text
Сократи каждый слайд до 1 главной фразы. Body text перенеси в caption или отдельный пост.
```

## GPT Image ломает русский текст

Варианты:

1. Сократить текст.
2. Генерировать картинку без текста, а текст добавить в редакторе.
3. Сделать edit pass.
4. Перегенерировать только проблемный слайд.

## Лицо получилось не похоже

Не выпускать как финал.

Проверить:

- свои ли reference images;
- достаточно ли разные ракурсы;
- нет ли слишком сложной позы;
- не перекрывает ли лицо текст/объект;
- нужен ли слайд вообще с лицом.

## Агент обещает слишком много

Исправление:

```text
Убери гарантии. Переформулируй как образовательный материал, опыт, наблюдение или гипотезу. Добавь risk note.
```

## Картинки не собираются через local CLI

Проверить:

```bash
gpt-image2-agent --help
```

Потом dry-run:

```bash
gpt-image2-agent "test image, no private data" --root . --dry-run --json
```

Если dry-run работает, но live нет - проблема в доступе, токене, лимитах, регионе, аккаунте или условиях сервиса. Не называйте это ошибкой carousel kit.
