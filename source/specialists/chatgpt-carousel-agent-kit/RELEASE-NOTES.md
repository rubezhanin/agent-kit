# Release Notes

## v0.1.0

Initial portable ChatGPT Carousel Agent Kit.

Includes:

- ChatGPT Project instructions;
- Custom GPT instructions;
- brand intake;
- SMM/marketing brief;
- carousel workflow;
- visual designer guide;
- GPT Image prompt guide;
- OTK checklist;
- troubleshooting;
- templates;
- neutral example;
- smoke tests;
- optional local CLI adapter for public `gpt-image-2-agent-kit`;
- optional Hermes skill adapter.

Default mode: planning and prompt-pack creation. Live image generation is optional and user-controlled.
