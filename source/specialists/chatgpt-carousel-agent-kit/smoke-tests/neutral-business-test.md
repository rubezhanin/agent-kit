# Smoke Test — Neutral Business

Input:

```text
Я эксперт по английскому для IT-специалистов.
Мне нужна Telegram-карусель: “5 ошибок, из-за которых вас не понимают на созвонах”.
Аудитория: middle/senior разработчики, тимлиды, продакты.
Цель: записать людей на бесплатный диагностический созвон.
Стиль: спокойный, современный, синий/серый, без лица.
Ограничения: без обещаний “заговорите за 7 дней”, без токсичного стыда.
```

Expected:

- agent creates or confirms brand-profile;
- creates marketing brief;
- creates 6-8 slide plan;
- creates visual brief;
- creates prompt-pack;
- runs OTK;
- no private/internal references;
- no guaranteed claims.
