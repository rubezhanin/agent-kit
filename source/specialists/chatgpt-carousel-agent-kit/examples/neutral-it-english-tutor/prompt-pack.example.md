# Prompt Pack — Example

## Global Style

Brand style: clean educational, modern, calm.
Colors: navy, light gray, white, muted blue accent.
Typography: large clean Cyrillic sans-serif.
Format: square Telegram carousel slide, 1280x1280.
People/reference policy: no people, no faces.
Negative global constraints: no toxic shaming, no fake logos, no tiny text, no random AI glow, no watermarks.

Prompt: Create a square Telegram carousel slide, 1280x1280. Large readable Russian headline: “5 ошибок, из-за которых вас не понимают на созвонах”. Visual metaphor: clean video call interface with blurred speech bubbles showing misunderstanding, no real app logos. Style: calm modern educational design, navy/light gray/white palette with muted blue accent, clean sans-serif typography, high contrast, 12% safe margins. Composition: headline top-left, visual call metaphor center-right, minimal background.
Negative constraints: no fake brand logos, no tiny paragraphs, no faces, no watermarks, no aggressive red, no chaotic clutter.
OTK risk: Cyrillic text accuracy.
