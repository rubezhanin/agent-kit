# Example Input

Я эксперт по английскому для IT-специалистов.

Тема карусели: `5 ошибок, из-за которых вас не понимают на созвонах`.

Аудитория: middle/senior разработчики, тимлиды, продакты.

Цель: записать людей на бесплатный диагностический созвон.

Стиль: спокойный, современный, синий/серый, без лица.

Ограничения: без обещаний “заговорите за 7 дней”, без токсичного стыда.
