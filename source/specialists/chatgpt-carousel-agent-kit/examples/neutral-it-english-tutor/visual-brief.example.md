# Visual Brief — Example

Global format: square Telegram carousel, 1280x1280. Optional Instagram adaptation: 1080x1350.
Global style: clean educational, calm, modern.
Colors: navy, light gray, white, muted blue accent.
Typography: large clean sans-serif, high contrast, short Russian headlines.
People/reference policy: no people, no faces, no identity references.
Safe-zone plan: keep all text and key objects inside 12% margin.
Global negative constraints: no fake logos, no real app UI, no tiny text, no random AI glow, no stock-photo people, no watermarks, no aggressive red.

## Slide 1

Text: `5 ошибок, из-за которых вас не понимают на созвонах`
Meaning job: open the pain and make the viewer self-check.
Visual metaphor: video call window with blurred speech bubbles.
Composition: headline top-left, abstract call card center-right, clean background.
Text zone: top-left reserved area, no overlap with visual.
Prompt notes: no real Zoom/Meet/Teams logos; use generic interface shapes.
OTK risk: Cyrillic headline must be readable.

## Slide 2

Text: `Проблема часто не в английском. А в структуре мысли.`
Meaning job: reframe the issue from vocabulary shame to thinking structure.
Visual metaphor: chaotic notes turning into three clear blocks.
Composition: before/after layout, left messy, right structured.
Text zone: top area, enough contrast.
Prompt notes: no tiny sticky-note text; only shapes.
OTK risk: avoid clutter.

## Slide 6

Text: `Мини-чеклист перед call`
Meaning job: give saveable value.
Visual metaphor: clean checklist card with three simple blank check rows.
Composition: checklist centered, headline top, blue accent checkmarks.
Text zone: headline + large checklist labels only if readable.
Prompt notes: if generated text fails, make checklist blank and add text manually.
OTK risk: text distortion.

## Slide 7

Text: `Хотите понять, где теряется ваша мысль? Напишите “CALL”`
Meaning job: CTA.
Visual metaphor: simple message bubble with `CALL`.
Composition: large CTA text, message bubble, calm background.
Text zone: centered.
Prompt notes: no promise of guaranteed improvement.
OTK risk: CTA must stay readable.
