# Carousel Brief — Example

Platform: Telegram + Instagram.
Format: square for Telegram, 4:5 optional for Instagram.
Goal: диагностический созвон.
Slide count: 7.
Core idea: проблема не в словарном запасе, а в ясности коммуникации.
Hook: `5 ошибок, из-за которых вас не понимают на созвонах`.
Arc: узнать себя -> понять механизм -> получить быстрый self-check -> записаться на диагностику.
CTA: `Сохраните чеклист и напишите “CALL”, если хотите диагностику речи на созвонах.`
Style direction: clean educational, blue/gray, no face.
Required gates: claims risk low; no external facts needed.
Assumptions: examples generalized.
