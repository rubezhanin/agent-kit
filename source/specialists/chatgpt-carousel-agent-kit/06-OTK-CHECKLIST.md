# 06 — OTK Checklist

Финальная проверка перед публикацией.

## Verdicts

```text
PASS
PASS_WITH_NOTES
FIX
REJECT
BLOCKED
```

## Marketing OTK

- Hook понятен за 2 секунды.
- Slide 2 усиливает боль/интерес, а не проваливает внимание.
- Есть причина листать дальше.
- Есть причина сохранить/переслать/ответить.
- CTA конкретный.
- Нет пустых общих фраз.
- Нет обещаний, которые нельзя доказать.

## Structure OTK

- Один слайд = одна мысль.
- Слайды идут логично.
- Нет повторов.
- Нет “простыни” текста.
- Финальный слайд полезен сам по себе.

## Visual OTK

- Текст читается на телефоне.
- Контраст достаточный.
- Safe margins соблюдены.
- Главный объект не спорит с текстом.
- Нет AI-мусора, странных рук, лиц, логотипов, воды, лишних символов.
- Стиль соответствует `brand-profile`.
- Нет случайного неона/роботов/stock-art, если это не запросили.

## Telegram OTK

- Square/landscape preview читается в ленте.
- Первый слайд работает как самостоятельная картинка.
- Заголовок крупный.
- Мелкий текст почти отсутствует.

## Instagram OTK

- 4:5 layout нормальный.
- Cover выдерживает 1:1 crop.
- CTA/final slide выдерживает 1:1 crop.
- Серия выглядит как одна система.

## Risk OTK

BLOCKED, если есть:

- медицинская/финансовая/юридическая гарантия без оснований;
- чужое лицо без разрешения;
- чужой закрытый стиль/брендбук;
- фейковые данные, цифры, отзывы, интерфейсы;
- секреты или приватная информация в prompt/receipt;
- утверждение, которое может навредить репутации пользователя.
