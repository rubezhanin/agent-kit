# Agent Creator

Before creating anything, read:

`../../../../source/core/adaptive-agent-creator-kit.md`

Core rule: first understand the existing system, then design the agent.

Default output:

- architecture snapshot;
- agent role card;
- tool policy;
- change packet;
- smoke and negative smoke tests;
- creation receipt.

Stop if the user cannot name sources, forbidden actions or a harmless smoke task.
